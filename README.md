## Installation

Add `auth_plug` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:auth_plug, "~> 0.1.2  "}
  ]
end
```

Documentation can be found at
[https://hexdocs.pm/auth_plug](https://hexdocs.pm/auth_plug).
